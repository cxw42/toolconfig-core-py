toolconfig - CLI interface¶

Get tool settings to apply to a particular file in a project

toolconfig [-h] [--version] [--ec-filename FILENAME] PATH [PATH ...]

toolconfig positional arguments¶

  • PATH - The absolute path to the file in the project (default: None)

toolconfig optional arguments¶

  • -h, --help - show this help message and exit

  • --version, -V - show program’s version number and exit

  • --ec-filename FILENAME, -f FILENAME - Alternative name for .editorconfig files (TESTING ONLY). Changes output format. (default: None)
